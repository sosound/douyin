# 抖音作品下载 Flask 应用 — 产品文档（v1.0）

> 版本：**v1.0**  
> 对应后端代码标签：`v1.0`  
> 最后更新时间：2026-02-28

---

## 一、产品概述

抖音作品下载 Flask 应用是基于 **DouK-Downloader Web API** 的一个轻量级 Web 前端，面向需要「**快速解析并下载单个抖音作品**」的用户。

通过浏览器访问本应用，用户只需粘贴抖音作品链接或分享口令，即可一键获取作品信息并下载无水印视频/图集。

---

## 二、目标用户与使用场景

- **自用备份**：创作者或普通用户希望把自己或他人的公开作品保存到本地。
- **素材整理**：短视频剪辑、运营人员需要批量收集素材（单个作品逐个下载）。
- **技术验证**：用于验证 DouK-Downloader 的 `/douyin/detail`、`/douyin/share` 接口是否工作正常。

本应用仅提供 **单个作品级别** 的解析与下载，不涉及账号批量下载、列表抓取等高级功能（这些由 DouK-Downloader 本体提供）。

---

## 三、核心能力（v1.0）

- **作品解析**
  - 支持直接输入以下形式的文本：
    - 完整作品链接（示例）  
      - `https://www.douyin.com/video/7201234567890123456`
      - `https://www.douyin.com/note/7201234567890123456`
      - `https://www.douyin.com/slides/7201234567890123456`
    - 仅作品 ID：  
      - `7201234567890123456`
    - 含短链的分享口令 / 分享文案（**v1.0 新增**）  
      - `5.30 n@q.RK ytR:/ 04/15 “我有明珠两颗 照亮青山万朵” # 一家四口  https://v.douyin.com/xxxxxx/ 复制此链接...`
      - `5.30 n@q.RK ... v.douyin.com/xxxxxx/ 复制此链接...`（即使没有 `http(s)://` 也可解析）
- **短链解析（v.douyin.com）**
  - 自动识别文本中出现的 `v.douyin.com/xxxxxx` 短链接；
  - 若输入中无法直接提取 19 位作品 ID，则自动调用 DouK-Downloader 的 `/douyin/share` 接口，将短链解析为完整作品链接；
  - 在解析后的完整链接中提取作品 ID 并继续后续流程。
- **作品详情展示**
  - 类型：视频 / 图集 / 实况
  - 作品 ID
  - 描述文案
  - 作者昵称 / 唯一 ID
  - 点赞数、评论数
- **无水印下载**
  - 调用 DouK-Downloader `/douyin/detail` 接口获取下载链接；
  - 由 Flask 后端 `/download` 路由代理下载，绕过防盗链；
  - 根据 `Content-Type` 自动推断文件扩展名（如 `mp4`、`jpg`、`png` 等）。
- **网络访问增强**
  - 支持可选的 **Cookie**：在抖音需要登录态时可填写；
  - 支持可选的 **HTTP 代理**：例如 `http://127.0.0.1:7890`，用于解决部分地区/网络限制。

## 四、系统架构与依赖

### 4.1 整体结构

- 前端：基于原生 HTML + CSS + JavaScript 的单页界面（文件：`templates/index.html`）
- 后端：Flask 应用（文件：`app.py`）
  - `POST /api/detail`：接收前端请求，调用 DouK-Downloader API，返回作品数据
  - `GET /download`：代理下载文件，处理防盗链与基础校验
- 底层解析 & 下载能力：由 **DouK-Downloader Web API** 提供：
  - `POST /douyin/share`
  - `POST /douyin/detail`

### 4.2 运行依赖

- Python >= 3.12
- Flask 及 httpx 等依赖（详见 `flask_app/requirements.txt`）
- DouK-Downloader Web API 服务：
  - 默认地址：`http://127.0.0.1:5555`
  - 需提前启动并保持可用

## 五、主要交互流程

### 5.1 用户操作流程

1. 启动 DouK-Downloader Web API（监听 `http://127.0.0.1:5555`）。
2. 在本仓库中进入 `flask_app` 目录，安装依赖并启动 Flask 应用：

   ```bash
   cd flask_app
   pip install -r requirements.txt
   python app.py
   ```

3. 浏览器访问 `http://127.0.0.1:5009`。
4. 在页面中完成以下操作：
   - 在「作品链接」输入框粘贴：
     - 抖音作品链接 / 作品 ID，或
     - 任意包含 `v.douyin.com/xxxxxx` 的分享口令/分享文案。
   - 如有需要，填写 Cookie 和代理地址。
   - 点击「获取作品数据」按钮。
5. 页面展示作品详情与下载按钮：
   - 若 `downloads` 为单个链接：展示「点击下载」；
   - 若为多个链接：展示「下载 1 / 下载 2 / ...」等多条下载按钮。
6. 点击下载按钮，浏览器通过 `/download` 路由获取文件并保存到本地。

## 六、接口设计（Flask 应用层）

### 6.1 `POST /api/detail`

**用途**：给前端页面提供统一的作品数据获取接口。

**请求体（JSON）**：

| 字段   | 类型   | 必填 | 说明                                 |
|--------|--------|------|--------------------------------------|
| link   | string | ✅   | 用户粘贴的作品链接 / 文本 / 分享口令 |
| cookie | string | ❌   | 可选，抖音 Cookie                    |
| proxy  | string | ❌   | 可选，HTTP 代理地址                  |

**处理逻辑（简要）**：

1. 校验 `link` 是否为空；
2. 若传入 `cookie`、`proxy`，保存到会话中（用于后续下载）；
3. 先尝试在 `link` 中提取 19 位作品 ID；
4. 若未提取到作品 ID：
   - 在文本中查找 `v.douyin.com/xxxxxx` 短链（支持无协议）；
   - 若存在，调用 DouK-Downloader 的 `/douyin/share` 接口解析短链，得到完整链接；
   - 再从解析后的链接中提取作品 ID；
5. 若仍无法得到作品 ID，则返回错误提示；
6. 使用作品 ID 调用 `/douyin/detail` 接口获取作品数据；
7. 统一包装返回 JSON 给前端页面。

**成功响应示例**（简化）：

```json
{
  "success": true,
  "message": "获取数据成功！",
  "detail_id": "7201234567890123456",
  "data": {
    "type": "视频",
    "id": "7201234567890123456",
    "desc": "作品描述",
    "nickname": "作者昵称",
    "downloads": [
      "https://...无水印视频下载地址..."
    ]
  }
}
```

### 6.2 `GET /download`

**用途**：代理 DouK-Downloader 返回的下载地址，避免直接暴露源站并解决防盗链问题。

**请求参数**：

| 参数 | 位置 | 必填 | 说明                  |
|------|------|------|-----------------------|
| url  | query| ✅   | DouK-Downloader 返回的下载链接 |

**核心行为**：

- 对 `url` 做安全校验：
  - 仅允许 `https` 协议；
  - 拒绝 `localhost` / `127.0.0.1` / `0.0.0.0` / `*.local` 等地址，防止 SSRF。
- 构造带有 Referer / User-Agent 等头的请求，传入可选 Cookie 与代理；
- 若响应为 HTML 页面，视为下载失败（可能链接过期或缺少 Cookie）；
- 根据内容类型推断文件扩展名，构造下载响应给浏览器。

## 七、错误提示与用户反馈

### 7.1 常见错误类型

- **「请输入作品链接」**
  - 未填写 `link` 字段，直接点击了「获取作品数据」。
- **「无法提取作品 ID，请确认链接格式正确」**
  - 输入的文本中既不包含合法的 19 位作品 ID，也没有可解析的 `v.douyin.com` 短链；
  - 或 DouK-Downloader `/douyin/share` 返回的链接中仍无法提取作品 ID。
- **「短链接解析失败，请检查 DouK-Downloader 是否运行」**
  - 解析短链时 `/douyin/share` 接口请求失败或返回异常；
  - 需确认 DouK-Downloader 服务是否启动、网络是否可达。
- **「获取数据失败」及具体错误信息**
  - `/douyin/detail` 请求失败或返回数据为空；
  - 页面会展示后端返回的 `message` 信息。
- **下载相关错误（通过 URL 参数返回到首页）**
  - 如「CDN 返回 HTML，可能链接已过期或需 Cookie」；
  - 如「下载失败: HTTP 状态码」。

### 7.2 用户可见反馈

- 成功时：绿色提示文案 + 作品详情卡片 + 下载按钮；
- 失败时：红色错误提示文案，帮助用户根据错误信息自行排查。

在 v1.0 版本中，Flask 应用已经实现了从「**输入任意形式的单个抖音作品链接或分享文案**」到「**解析作品信息并下载无水印内容**」的完整闭环，并特别增强了对 **分享口令 + v.douyin.com 短链** 这一常见使用场景的支持，适合作为 DouK-Downloader 的配套可视化工具使用。
