# 抖音去水印下载（Galaxy 直连）— 快捷指令步骤说明

> 按下列步骤在 iPhone「快捷指令」中从零搭建「抖音去水印下载」快捷指令，直连 GalaxyStream 接口，无需自建后端。

---

## 常量（建议先设为变量便于修改）

| 变量名 | 说明 | 示例值 |
|--------|------|--------|
| `BaseURL` | GalaxyStream API 根地址 | `http://source.galaxystream.online:5555` 或 `https://...` |

---

## 步骤 1：获取输入文本

- **动作**：`从输入获取文本` 或 `获取剪贴板`。
- **逻辑**：若有「共享的输入」则用输入文本，否则用剪贴板内容。
- **结果**：得到变量 `输入文本`。

---

## 步骤 2：尝试从文本中提取作品 ID（19 位数字）

- **动作**：`匹配文本`（Match Text）。
  - **源文本**：`输入文本`。
  - **匹配模式**：正则 `\d{19}`（或 Shortcuts 中若不支持正则，可用「包含」先判断是否含 `douyin.com`，再交给下一步解析）。
- **结果**：
  - 若匹配到，将匹配结果存为 `detail_id`，**跳转到步骤 4**。
  - 若未匹配到，继续步骤 3。

> 部分系统「匹配文本」支持「正则表达式」；若不支持，可省略本步，直接进入步骤 3 用分享接口解析。

---

## 步骤 3：解析短链（未直接得到 ID 时）

- **动作**：`获取 URL 的内容`（Get Contents of URL）。
  - **URL**：`BaseURL/douyin/share`（例如 `http://source.galaxystream.online:5555/douyin/share`）。
  - **方法**：`POST`。
  - **请求体**：选择「JSON」。
  - **JSON**：  
    `{"text": "<输入文本>"}`  
    （将上一步的「输入文本」代入）
- **结果**：解析返回的 JSON，取键 `url` 的值（完整链接，如 `https://www.douyin.com/video/7201234567890123456`）。
- **再匹配**：对 `url` 再次用 `\d{19}` 匹配，得到 `detail_id`。
- **若仍无 19 位数字**：显示「无法解析链接，请确认已复制抖音分享链接」并结束。

---

## 步骤 4：获取作品数据

- **动作**：`获取 URL 的内容`。
  - **URL**：`BaseURL/douyin/detail`（例如 `http://source.galaxystream.online:5555/douyin/detail`）。
  - **方法**：`POST`。
  - **请求体**：JSON。
  - **JSON**：  
    `{"detail_id": "<detail_id>"}`  
    （将前面得到的 19 位作品 ID 代入）
- **结果**：解析返回的 JSON。

---

## 步骤 5：取出下载链接

- **动作**：`获取字典值` / `获取字典项`。
  - 从响应中依次取：`data` → `downloads` → **第一项**（索引 0）。
- **结果**：得到变量 `下载链接`（一个 URL 字符串）。
- **错误处理**：若 `data` 为空或 `downloads` 为空，提示「获取下载链接失败」并结束。

---

## 步骤 6：下载并保存

- **动作**：`获取 URL 的内容`（选择「文件」/下载）或「下载」类动作。
  - **URL**：`下载链接`。
- **后续**：
  - **保存到相册**：使用「存储到相册」。
  - **保存到文件**：使用「存储文件」选择位置。

---

## 请求 / 响应示例（便于调试）

**1. POST /douyin/share**

```json
// 请求
{ "text": "8.53 复制打开抖音，看看【xxx】的精彩作品 https://v.douyin.com/xxxxx/" }

// 响应（成功）
{
  "message": "请求链接成功！",
  "url": "https://www.douyin.com/video/7201234567890123456",
  "params": { "text": "...", "proxy": "" },
  "time": "..."
}
```

**2. POST /douyin/detail**

```json
// 请求
{ "detail_id": "7201234567890123456" }

// 响应（成功）
{
  "message": "获取数据成功！",
  "data": {
    "type": "视频",
    "id": "7201234567890123456",
    "desc": "...",
    "downloads": ["https://xxx.douyincdn.com/...无水印视频地址"],
    ...
  },
  "params": { ... },
  "time": "..."
}
```

快捷指令只需使用 `data.downloads` 的第一个元素作为下载地址即可。
