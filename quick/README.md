# iPhone 快捷指令集成（Quick） - Douyin Downloader

> 目录：`quick/`  
> 目标：在 iPhone 上通过「快捷指令」直接完成抖音作品解析与下载（复用 `flask_app` 的能力）。

---

## 1. 整体思路

- **后端能力**：由 `flask_app` 提供，已经支持：
  - 从任意抖音作品链接 / 文本 / 分享口令中提取作品 ID；
  - 自动解析 `v.douyin.com` 短链；
  - 返回作品详情与下载地址；
  - 代理下载文件。
- **iPhone 快捷指令**：作为前端入口，完成：
  - 从「共享菜单」或「剪贴板」接收抖音分享内容；
  - 调用本机局域网中的 Flask 应用（HTTP 请求）；
  - 解析 JSON，拿到下载链接；
  - 在手机端直接下载 / 保存到相册 / 保存到文件。

本目录用于存放与快捷指令相关的设计文档、示例配置等（不必直接存放 `.shortcut` 二进制文件）。

---

## 2. 依赖条件

1. **后端服务已运行**
   - 已按 `flask_app/README.md` 启动 Flask 应用：
     - DouK-Downloader Web API：`http://127.0.0.1:5555`
     - Flask Web：默认 `http://127.0.0.1:5009`
   - iPhone 能访问到运行 Flask 的电脑（同一局域网，使用电脑局域网 IP 替代 `127.0.0.1`）。

2. **iPhone 环境**
   - iOS 16+（推荐）
   - 已安装 Apple 官方「快捷指令」App
   - 在「设置 > 快捷指令」中允许「允许不受信任的快捷指令」（首次需要打开）。

---

## 3. 快捷指令 v1.0 设计（草案）

### 3.1 使用方式

- 在抖音 App 中，点击「分享」→「复制链接」；
- 或者在分享菜单中选择「快捷指令」扩展（如有配置）；
- 触发「抖音去水印下载（本地服务器）」快捷指令：
  - 自动读取剪贴板 / 分享输入；
  - 调用本机 Flask 服务接口；
  - 完成下载并保存到相册/文件。

### 3.2 后端接口约定

快捷指令将直接复用 `flask_app` 提供的接口：

- **获取作品数据**：`POST http://<电脑局域网IP>:5009/api/detail`
  - 请求体（JSON）：
    - `link`: 分享内容（文本）—— 由快捷指令传入（剪贴板或分享输入）
    - `cookie`: 可留空
    - `proxy`: 可留空，除非需要代理
  - 重要响应字段：
    - `success`: 是否成功
    - `message`: 文本提示
    - `data.downloads`: 下载链接列表

- **下载文件**（可选两种策略）：
  1. 由 **快捷指令直接下载 `data.downloads[0]`**（推荐简单场景）；
  2. 通过 Flask `/download?url=...` 代理下载，再由手机保存（需要额外 URL 拼接）。

在 v1.0 设计中，可以先采用 **方案 1：快捷指令直接下载原始链接**，如遇到防盗链/访问问题，再切到方案 2。

## 4. 快捷指令步骤示意（伪流程）

> 以下为逻辑说明，可在 iPhone 上按步骤手动创建。

1. **获取输入文本**
   - 从「共享的输入」获取文本；如果为空，则从剪贴板获取文本；
2. **构造 HTTP 请求**
   - 动作：`获取网址的内容` / `获取 URL 的内容`；
   - 方法：`POST`；
   - URL：`http://<电脑IP>:5009/api/detail`
   - 请求体类型：`JSON`；
   - JSON 字段：
     - `link` = 第 1 步得到的文本；
3. **解析 JSON 响应**
   - 动作：`获取字典值` / `获取字典项`；
   - 检查 `success` 是否为 `true`；
   - 取 `data` → `downloads` → 第一个元素，记为 `下载链接`；
4. **下载文件**
   - 动作：`下载` / `获取网址的内容`（以文件形式）；
   - URL：`下载链接`；
   - 完成后可选：
     - `保存到相册`；
     - 或 `存储文件`。

后续可以在本目录中进一步细化为「图示步骤」或「导出 JSON 模板」。
